# 🍜 Restaurant Tracker

A personal app to track saved restaurants, check what's open right now, 
and get smart suggestions based on cuisine, mood, and time of day.

## Features (in progress)
- 📸 Upload Apple Maps screenshots to add restaurants automatically
- 🕐 Real-time "open now" filtering from your saved list
- 🍣 Filter by cuisine, price, and neighborhood
- 🤖 ML-powered suggestions based on your preferences

## Tech Stack
- Frontend: React Native / React
- Backend: Node.js + Express
- Database: Supabase
- APIs: Google Places, Anthropic Claude
- ML: TBD (preference learning model)
